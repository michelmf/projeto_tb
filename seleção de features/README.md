# Seleção de atributos

Em um conjunto de dados, muitos dos atributos presentes podem ser irrelevantes para a predição da classe a qual o exemplo a ser avaliado pertence. A tarefade seleção de atributos tem como objetivo reduzir o espaço de busca das soluções,removendo variáveis irrelevantes e/ou redundantes ao problema, ou seja, definir um subconjunto de atributos essenciais para uma predição satisfatória das classes do problema. Para $n$ atributos existentes, existem $n^{2}$ possíveis combinações de atributos. Uma busca exaustiva do melhor subconjunto é normalmente custosa e, namaioria dos casos, proibitiva. Assim, é de interesse dispor de formas mais eficazes de se obter o melhor subconjunto de atributos que levem os classificadores a uma melhor performance possível. A busca por um subconjunto de informações mais relevantes para o problema reduz o custo computacional do treinamento dos modelos,e simplifica de forma geral a interpretação dos modelos criados.
